题目描述

实现函数 ToLowerCase()，该函数接收一个字符串参数 str，并将该字符串中的大写字母转换成小写字母，之后返回新的字符串。

示例 1：

输入: "Hello"
输出: "hello"

示例 2：

输入: "here"
输出: "here"

示例 3：

输入: "LOVELY"
输出: "lovely"

C++代码实现

class Solution {
public:
    string toLowerCase(string str) {
        size_t len = str.length();
        for(size_t i = 0; i < len; i++){
            if(str[i] >= 'A' && str[i] <= 'Z'){
                str[i] = str[i] - 'A' + 'a';
            }
        }
        return str;
    }
};

代码性能

算是比较简单的一道题，主要思路就是根据ASCII编码进行计算。
